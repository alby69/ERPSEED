# ROADMAP Generale ERPSeed

## Visione d'Insieme
ERPSeed mira a diventare il punto di riferimento per la creazione rapida di gestionali aziendali, unendo la flessibilità del low-code alla potenza dell'intelligenza artificiale.

## 🎯 Obiettivi Strategici
1. **Disaccoppiamento Totale**: Backend e Frontend devono poter evolvere indipendentemente comunicando via API.
2. **AI-First**: L'assistente AI deve essere integrato in ogni fase della costruzione dell'ERP.
3. **Semplicità (KISS)**: Mantenere il core snello e facile da estendere.

## 🚀 Milestone Principali
- [x] Consolidamento Architettura Modulare (Backend)
- [x] Lancio del Visual Builder (Frontend)
- [ ] Integrazione completa AI per generazione automatica UI/Modelli
- [ ] Marketplace dei moduli (Seed Ecosystem)

## 🔮 Sviluppi Futuri (Idee Globali)
- **ERPSeed Cloud**: Versione SaaS per installazioni istantanee.
- **Mobile App**: Client nativo per iOS/Android basato su React Native.
- **DAO per il Marketplace**: Gestione comunitaria della qualità e dei proventi dei moduli nel marketplace.
