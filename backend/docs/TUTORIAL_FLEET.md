# Tutorial: Building your first project with ERPSeed

Welcome to ERPSeed! This tutorial will guide you through the process of creating a simple **Fleet Management** system from scratch using the No-Code Builder and the AI Assistant.

## Prerequisites
- ERPSeed installed and running (Backend on port 5000/5002, Frontend on port 5173).
- Logged in as an administrator.

---

## Step 1: Create a New Project

1. Go to the **Projects** page.
2. Click **New Project**.
3. Fill in the details:
   - **Name**: `fleet_management`
   - **Title**: `Fleet Management`
   - **Description**: `System to manage company vehicles and maintenance.`
4. Click **Create**.
5. Select the newly created project to enter the **Project Dashboard**.

---

## Step 2: Define your first Model (Vehicle)

We need a model to store vehicle information.

### Option A: Using the Visual Builder (Manual)
1. In the Project Dashboard, go to **Models**.
2. Click **New Model**.
3. **Name**: `Vehicle`, **Table**: `vehicles`.
4. Add the following fields:
   - `Plate` (String, Required, Unique)
   - `Model` (String, Required)
   - `Brand` (Select: Fiat, Ford, Tesla, etc.)
   - `PurchaseDate` (Date)
   - `Mileage` (Integer)
5. Click **Save and Publish**.

### Option B: Using the AI Assistant (Fast)
1. Open the **AI Assistant** (usually a floating bubble or a side panel).
2. Type: `"Create a model for Vehicles with fields: plate (unique), brand, model, purchase date, and mileage."`
3. Review the execution plan generated by the AI.
4. Click **Apply Changes**.

---

## Step 3: Create a Maintenance Model (Relational Data)

Now let's track maintenance for these vehicles.

1. Create a new model: `Maintenance`.
2. Add fields:
   - `Description` (Text)
   - `Date` (Date, Required)
   - `Cost` (Decimal)
   - `Vehicle` (**Relation** to `Vehicle` model, Required).
3. In the **Vehicle** relation options, set:
   - **Target Model**: `Vehicle`
   - **Label Field**: `Plate` (this is what you will see in the dropdown).
4. Click **Save and Publish**.

---

## Step 4: Configure Views

ERPSeed automatically creates default list and form views, but let's customize them.

1. Go to **Views** -> **Vehicle List**.
2. Drag and drop columns to reorder.
3. Add a **Filter** to show only vehicles with mileage > 100,000.
4. Save the view.

---

## Step 5: Test your Application

1. Go to the **Application** menu (the "play" icon or the project title in the sidebar).
2. You should see **Vehicles** and **Maintenance** in the menu.
3. Add a new Vehicle:
   - Plate: `AB123CD`
   - Brand: `Ford`
   - Model: `Transit`
4. Add a Maintenance record for this vehicle.

---

## Step 6: Add an Automation (Optional)

Let's notify the owner when a maintenance cost exceeds $1,000.

1. Go to **Workflows**.
2. Create **New Workflow**: `High Maintenance Alert`.
3. **Trigger**: `On Record Created` for model `Maintenance`.
4. **Condition**: `{{record.cost}} > 1000`.
5. **Action**: `Send Email` or `System Notification`.
6. Save and Enable.

---

## Summary
Congratulations! You've built a multi-model relational application with custom views and automation without writing a single line of code.
